# Minecraft-Anti-AFK-Bot
Minecraft Bot Which Can Keep Your Server Like Aternos Awake For 24/7
----------------------------------------------------------------------

 Requirements
--------------

>> Install dependencies first: type 'npm install mineflayer' in the terminal this will installs mineflayer (Mineflayer Required To Run The Code)
>> Change the server ip and port to your original server ip and port where you want the bot to join (it wont work if the ip and port are not correct)
>> Then in terminal type 'node bot.js' or 'node YOUR_FILE_NAME.js'
